enum pwm_mode_enum_t

Specifies the PWM mode to use.

Enumerator
FAST_PWM 

The duty cycle starts with the period. This can have twice as high frequencies compared to the centered mode.

CENTERED_PWM 

The duty cycle is centered around the middle of the period, but not as the frequency changes. It is centered around the middle of the period even if the frequency changes only for Timer1 and Timer3

u32  u32_frequency 
)

Sets the frequency to be used in PWM mode and updates duty cycle accordingly

Remarks
This changes the frequency for both channel outputs of a timer if using Timer1 or Timer3
Actual timer frequency may differ from the expected one due to division rounding.
Precondition
PWM must be initialized in either FREQUENCY_VARIABLE or FULLY_VARIABLE signal type.
Parameters
[in]s_pwmtimer peripheral to use
[in]u32_frequencyfrequency in Hz to set
