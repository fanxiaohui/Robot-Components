◆ timer_prepareSynchronisedStart()

void timer_prepareSynchronisedStart ( )

Prepares all timers to be started at the same time.

Precondition
Must be called after all timers were initialized (with timer_init).
Remarks
Calling this freezes all timers so be careful when using it if one timer is already running.
Calling timer_start after this has no immediate effect but is required before a synchronised start.
