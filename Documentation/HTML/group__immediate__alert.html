void immediate_alert_hook ( alert_level_t  level)

Callback on immediate alert.

This function is called when an immediate alert characteristic value is updated. Implement this function to have your application behave according to the alert level of the immediate alert.

Parameters
level: Level of the immediate alert (see alert_level_characteristic to view possible values).

Function to handle reception of data for immediate_alert.

Call this function on the ACI_EVT_DATA_RECEIVED event. This function checks if the data received is for immediate_alert. If so it triggers the immediate alert with the received level.

Parameters
pipe_numPipe which has received data.
bufferPointer to the received data.
