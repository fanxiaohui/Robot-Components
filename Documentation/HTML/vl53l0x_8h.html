Detailed Description

VL53L0X distance sensor.

Supports only master mode without interrupts. Basic flow:

  1. Initialize and start a timer. Make it call vl53l0x_incrementTimeoutCounter every millisecond.
  2. Initialize a vl53l0x_struct_t.
  3. Pass it to vl53l0x_init.
  4. Call vl53l0x_start.
  5. Call vl53l0x_startContinuous to start continuous measurements.
  6. Call vl53l0x_readRangeContinuous to get distance measurements. Measurement hasn't finished yet if return value is 0xFFFF.
  • Optionally you could use vl53l0x_setMode with one of the vl53l0x_mode_enum_t modes to change the measurement duration, accuracy or max range.
  • To stop the sensor (for power saving for instance) call vl53l0x_stop. Remember that continuous ranging has to be started in order to take measurements after calling vl53l0x_start.

High accuracy mode. High accuracy, 600mm max range, 200ms measurement time

High range mode. Low accuracy, 1200mm max range, 33ms measurement time

High speed mode. Low accuracy, 600mm max range, 20ms measurement time
