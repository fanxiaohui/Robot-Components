void ancs_notification_evt_rcvd ( uint8_t  pipe_num,
uint8_t *  buffer 
)

Function to handle reception of data for ANCS.

Call this function on the ACI_EVT_DATA_RECEIVED event. This function checks if the data received is for ANCS. If so it prints the received data to the Serial interface.

Parameters
pipe_numPipe which has received data.
bufferPointer to the received data.
