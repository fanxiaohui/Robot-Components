◆ wdg_enableReset()

void wdg_enableReset ( )

Enables the watchdog counter and sets it to reset the microcontroller if it overflows

Remarks
wdg_enableInterrupt can be called while Reset mode is enabled to trigger an interrupt prior to the actual reset. Note that doing so does not prevent the actual reset once the watchdog overflows.
Warning
Remember to call wdg_feed to prevent a microcontroller reset.
