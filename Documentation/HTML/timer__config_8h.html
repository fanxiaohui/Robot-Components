Detailed Description

Timer configuration.

Specifies the following:

  • which Timer peripherals are used (define only if used in timer mode)
  • if interrupt modes are used These defines are used for code size and memory usage reduction, so set the defines accordingly.
    Remarks
    If using other modes for the timers, their specific config files handle definitions accordingly and no change has to be made to this file
