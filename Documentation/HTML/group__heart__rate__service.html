void heart_rate_pipes_updated_evt_rcvd ( aci_state_t *  aci_stat,

)

Function to check received data.

Call this function each time data is received (on ACI_EVT_DATA_RECEIVED on the control point pipe ). If the control point is received with the bits indicating to reset Expended Energy, then hook_for_resetting_energy_expended will be called.

Parameters
pipe_numPipe number.
bufferThe received message.
