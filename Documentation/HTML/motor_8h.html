Detailed Description

Motor features.

Author
Adrian Grosu
Version
1.0
Date
31.10.2017

Motor driver that supports 2 motors which are controlled with a digital output and a PWM signal for each motor

  1. Initialize a motor_struct_t. You do not need to initialize the structure members beforehand, you only have to declare them and set their values.
  2. Pass it to motor_init.
  3. Call motor_start.
  4. Call motor_speed when needed to change the speed.
  5. Call motor_direction when needed to change the direction.
  • You can call motor_individualDirSpeed to adjust the speed and direction for each of the motors individually
  • You can stop the motors by calling motor_stop.
    Remarks
    motor_individualDirSpeed only supports FORWARD and BACKWARD values for motorDirection.
