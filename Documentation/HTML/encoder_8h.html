Detailed Description

Encoder features.

Author
Adrian Grosu
Version
1.0
Date
31.10.2017

Supports encoders with 2 pin outputs

  1. Initialize a encoder_struct_t. You do not need to initialize the structure members beforehand, you only have to declare them and set their values.
  2. Pass it to encoder_init.
  3. Call encoder_start.
  4. Call encoder_getCounter when needed to check the number of impulses
  • You can reset the counter by calling encoder_resetCounter.
  • You can stop the encoders by calling encoder_stop.
    Remarks
    encoder_stop doesn't reset the value of the counter, it only disables the interrupts.
